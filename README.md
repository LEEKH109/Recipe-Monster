## RecipeMonster는
레시피 공유를 위한 사이트를 제작하기 위한 개인 프로젝트입니다.
React-TypeScript를 이용한 Client 제작을 계획 중입니다.
Server는 Express.js를 이용한 간단한 Server 와 Firebase를 이용한 방법 중 고민 중입니다.
